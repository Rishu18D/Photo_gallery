# Photo Gallery & Image Upload

Welcome to our Photo Gallery & Image Upload project! This web application allows users to upload images from their local system or capture images using their camera, creating a beautiful gallery to showcase their memories.

## Features

- Upload images from your local system
- Capture images using your device's camera
- Beautifully display images in a gallery format
- Delete images from the gallery
- Responsive design for seamless viewing on different devices

## Technologies Used

- **Frontend:**
  - HTML5
  - CSS3 (including Bootstrap for styling)
  - JavaScript (Vanilla JS)

- **Backend:**
  - Node.js
  - Express.js
  - Cloudinary (for image storage)
